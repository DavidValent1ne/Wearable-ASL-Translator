# Wearable-ASL-Translator
This project will be for creating a wearable ASL translator. This will be a glove covered in flex sensors, hall effect sensors, IMUs, and tilt sensors feeding into an ESP32. The ESP32 will take the data, format it, and send to a PI Zero which will contain a trained time-series clustering model.
